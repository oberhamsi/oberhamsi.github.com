rp password

rp password
rp password reset

Description

rp password allows you to change you registry password, or to request a password reset in case you have forgotten your password.

In case you know your registry password and want to change it, do rp password without any arguments. rp will ask you for your existing password, and the new one. If everything is correct, your registry password will be changed.

In case you forgot the password of your registry account, use rp password reset. This requires you to specify the email address used when creating your registry account. You should then receive an email with a password reset token and further instructions on how to use that token to reset your password.
