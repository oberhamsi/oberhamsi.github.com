rp list

rp list [options]

Options

-g/--global List global packages
-v/--verbose Display verbose information

Description

rp list displays the installed packages, either the locally installed ones or the globally installed packages (if -g or --global option is given).

If the -v/--verbose flag is given, rp list displays verbose information for each package (the installation directory of the package, the package description, author and any dependencies of the package).
