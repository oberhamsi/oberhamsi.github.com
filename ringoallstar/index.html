Ringo & His Band help you build websites on top of RingoJs. It pulls together a couple of Ringo packages to provide templating, a view engine and an ORM.

What's cool about it?

If you are a Java developer and want to give a dynamic language a shot, VirtueJs makes a lot of sense because we have good Java integration.

If you are a seasoned JavaScript developer, you'll enjoy the modern JavaScript as well as Ringo's extensive standard library and tools.

{% extends 'base.html' %}
{% block title %}Example Site{% endblock %}
{% block content %}
  <ul>
  {% for user in users %}
    <li><a href="{{ user.url }}">{{ user.username }}</a></li>
  {% endfor %}
  </ul>
{% endblock %}
