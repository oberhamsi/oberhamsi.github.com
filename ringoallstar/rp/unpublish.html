rp unpublish

rp unpublish <name> [version]

Description

rp unpublish removes a specific version or a whole package including all of its versions from the registry.

You need to be a package owner to do that. This command should only be used in case of erronous package publications. Obviously it's a bad idea to remove a whole package from the registry, unless you can be sure it's never been installed anywhere.
